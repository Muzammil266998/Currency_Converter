# Online Currency Converter

A free, responsive online currency converter to convert USD, EUR, GBP, INR, PKR, AED, SAR, and more. Quick, accurate, and simple UI.

## Features
- Real-time currency conversion using fixed rates
- Quick pair buttons: USD→PKR, EUR→USD, INR→USD, etc.
- Responsive layout with sidebar for ads
- SEO-friendly meta tags for better Google ranking
- AdSense ready placeholders

## Usage
1. Enter the amount
2. Select From and To currencies
3. Click Convert
4. See the result instantly
